# Info

Made during 2025 as part of Humanités Numériques curriculum (see enssib.fr)

# Install

Using php 8.1.2.

```
git clone https://github.com/bprunevi/HNMaps
cd HNMaps
php -S 127.0.0.1:8889
```

Then you may go to http://0.0.0.0:8000/ to see results.
